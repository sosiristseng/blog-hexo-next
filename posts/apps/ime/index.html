Typing Chinese in Linux.

Fcitx

Fcitx is a lightweight input method framework aimed at providing environment independent language support for Linux. The development energy is mainly focused on the new version 5 release.

  • By pacman
    1

    sudo pacman -S fcitx5-im fcitx5-chewing fcitx5-material-color
  • By apt (Ubuntu 21.04+)
    1

    sudo apt install fcitx5 fcitx5-chewing  citx5-material-color
If fcitx5 does not work properly

Attach these lines to ~/.xprofile or ~/.profile.

1
2
3
4

export INPUT_METHOD=fcitx5
export GTK_IM_MODULE=fcitx5
export QT_IM_MODULE=fcitx5
export XMODIFIERS=\@im=fcitx5

ibus

ibus is an input method framework using DBUS. ibus integrates with gnome.

  • By apt
    1

    sudo apt install ibus ibus-chewing
  • By pacman
    1

    sudo pacman -S ibus ibus-chewing
If ibus does not work properly

Attach the following lines to ~/.xprofile or ~/.profile.

1
2
3
4
5

# ~/.xprofile
export GTK_IM_MODULE=ibus
export QT_IM_MODULE=ibus
export XMODIFIERS=@im=ibus
ibus-daemon -drx
Change ibus system tray icon color

Using dconf1

1

gsettings set org.freedesktop.ibus.panel xkb-icon-rgba ${COLOR}

  1. https://wiki.archlinux.org/index.php/IBus#Systray_language_icon_color ↩︎
