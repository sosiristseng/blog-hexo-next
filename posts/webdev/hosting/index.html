Hosting static websites by:

  • GitHub pages
  • GitLab pages
  • Netlify
  • Cloudflare pages

GitHub pages

Github pages docs about creating a website from a GitHub repository.

  • You can find GitHub pages in the repository settings.
  • Fast connection through its CDN infrastructure.

GitLab pages

GitLab pages docs about creating a website from a GitLab repository.

  • Pages is set through pages artifacts
  • Custom domains with SSL/TLS cert.
  • Access control enalbe private pages requiring logins.

Netlify

Netlify sources GitHub / GitLab repositories to build and host websites.

  • Build, deploy, and manage websites
  • Fast via the CDN structure
  • Custom domains
  • Preview the result in your development branch
  • Rollback

Cloudflare pages

Cloudflare pages sources GitHub repositories (GitLab ones are WIP) to build and host websites.

  • Fast via the CDN structure
  • Unlimited collaborates for free
  • Preview the result in your development branch
  • Built-in, free web analytics
